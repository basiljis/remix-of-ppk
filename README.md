# Система автоматизации ППК

Автоматизированная система психолого-педагогического консилиума для образовательных организаций.

## Возможности

- Создание и ведение протоколов ППК
- Управление чек-листами оценки
- База данных образовательных организаций
- Статистика и отчетность
- Экспорт документов

## Разработка

Система построена на основе современных веб-технологий:
- React + TypeScript
- Tailwind CSS
- Supabase (база данных)
- Vite (сборщик)

## Развертывание

Для развертывания системы:

1. Установите зависимости: `npm install`
2. Настройте переменные окружения
3. Запустите разработческий сервер: `npm run dev`
4. Соберите продакшн версию: `npm run build`
